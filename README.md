# SQL Magic with SQLite in Jupyter Notebook

## Description
This project demonstrates how to use SQL Magic in Jupyter Notebook to
interact with an SQLite database without writing Python DB-API code.

## Concepts Covered
- Jupyter Magic Commands (% and %%)
- SQL Magic using ipython-sql
- SQLite database connection using %sql
- Creating and querying tables using %%sql

## Tools Used
- Python
- Jupyter Notebook
- SQLite
- ipython-sql

## Use Case
Useful for learning SQL inside Jupyter Notebooks and quick database exploration
for data analysis and education purposes.
